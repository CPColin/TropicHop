TropicHop is an adaptation of a previous project, which was an adaptation of a
copyrighted board game and, thus, had to be taken down. Only minor effort was
spent creating the new board layout.

Instructions for running:

Double-clicking the JAR or running it as below should work fine:

java -jar tropichop.jar

Or run the com.crappycomic.tropichop.main.Main class.
